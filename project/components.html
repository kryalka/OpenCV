Основные компоненты библиотеки OpenCV

OpenCV предоставляет мощные инструменты для работы с изображениями: загрузка, обработка, отображение и сохранение.

Загрузка изображений

Для загрузки изображений используется функция cv2.imread(), которая преобразует изображение в массив NumPy.

Синтаксис cv2.imread()

cv2.imread(filename, flags)
  • cv2.IMREAD_COLOR — цветное изображение (по умолчанию).
  • cv2.IMREAD_GRAYSCALE — изображение в оттенках серого.
  • cv2.IMREAD_UNCHANGED — сохранение альфа-канала.

Примеры загрузки изображений

1. Цветное изображение

import cv2

image = cv2.imread('example1.jpg', cv2.IMREAD_COLOR)
cv2.imshow('Загруженное изображение', image)
cv2.waitKey(0)
cv2.destroyAllWindows()

2. Изображение в оттенках серого

gray_image = cv2.imread('example1.jpg', cv2.IMREAD_GRAYSCALE)
cv2.imshow('Оттенки серого', gray_image)
cv2.waitKey(0)
cv2.destroyAllWindows()

3. Изображение с альфа-каналом

image_with_alpha = cv2.imread('example_with_alpha.png', cv2.IMREAD_UNCHANGED)
cv2.imshow('Альфа-канал', image_with_alpha)
cv2.waitKey(0)
cv2.destroyAllWindows()

Отображение изображений

Для отображения используется cv2.imshow(). Для Google Colab применяйте Matplotlib:

import cv2
import matplotlib.pyplot as plt

image = cv2.imread('example1.jpg')
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
plt.imshow(image_rgb)
plt.axis('off')
plt.show()

Сохранение изображений

Примеры сохранения

cv2.imwrite('output_90.jpeg', image, [cv2.IMWRITE_JPEG_QUALITY, 90])
cv2.imwrite('output_with_alpha.png', image_with_alpha)
