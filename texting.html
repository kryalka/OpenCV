Добавление текста и форм в OpenCV

Добавление текста

Для добавления текста на изображение в OpenCV используется функция cv2.putText.

Описание и параметры

  • img: изображение, на которое будет добавлен текст.
  • text: текст, который нужно добавить.
  • org: координаты начала текста (x, y).
  • font: шрифт текста (например, cv2.FONT_HERSHEY_SIMPLEX).
  • fontScale: масштаб шрифта.
  • color: цвет текста (BGR, например, (255, 0, 0) для синего).
  • thickness: толщина линии текста.
  • lineType: тип линии (обычно cv2.LINE_AA для сглаживания).

# Загружаем изображение
image = cv2.imread("example8.jpg")

if image is None:
    print("Ошибка: изображение не загружено.")
else:
    # Добавляем текст
    text = "Mya"
    org = (75, 100)  # Координаты начала текста
    font = cv2.FONT_HERSHEY_SIMPLEX
    fontScale = 1
    color = (255, 0, 0)  # Синий цвет
    thickness = 2

    cv2.putText(image, text, org, font, fontScale, color, thickness, cv2.LINE_AA)

    # Сохраняем изображение
    cv2.imwrite("text_result.jpg", image)

    # Отображаем результат с использованием cv2_imshow
    cv2_imshow(image)

Добавление форм

В OpenCV можно добавлять различные формы: прямоугольники, круги, линии и многое другое. Рассмотрим основные функции.

Прямоугольник

Функция cv2.rectangle добавляет прямоугольник.


cv2.rectangle(image, pt1, pt2, color, thickness)
# pt1: координаты верхнего левого угла
# pt2: координаты нижнего правого угла
# color: цвет (например, (0, 255, 0) для зеленого)
# thickness: толщина линии (-1 для заливки)

Пример:


# Загружаем изображение
image = cv2.imread("example8.jpg")

if image is None:
    print("Ошибка: изображение не загружено.")
else:
    color = (255, 0, 0)  # Синий цвет
    thickness = 2
    pt1 = (100, 100)
    pt2 = (500, 500)
    cv2.rectangle(image, pt1, pt2, color, thickness)
    # pt1: координаты верхнего левого угла
    # pt2: координаты нижнего правого угла
    # color: цвет (например, (0, 255, 0) для зеленого)
    # thickness: толщина линии (-1 для заливки)

    # Сохраняем изображение
    cv2.imwrite("text_result.jpg", image)

    # Отображаем результат с использованием cv2_imshow
    cv2_imshow(image)

Круг

Функция cv2.circle добавляет круг.


cv2.circle(image, center, radius, color, thickness)
# center: координаты центра круга
# radius: радиус круга
# color: цвет (например, (0, 0, 255) для красного)
# thickness: толщина линии (-1 для заливки)

Пример:


# Загружаем изображение
image = cv2.imread("example8.jpg")

if image is None:
    print("Ошибка: изображение не загружено.")
else:
    color = (255, 0, 0)  # Синий цвет
    thickness = 2
    
    center = (260, 260)  # Координаты центра круга в формате (x, y)
    radius = 75
    
    # Рисуем круг
    cv2.circle(image, center, radius, color, thickness)

    # Сохраняем изображение
    cv2.imwrite("gaga.jpg", image)

    # Отображаем результат с использованием cv2_imshow
    cv2_imshow(image)

Линия

Функция cv2.line добавляет линию.


cv2.line(image, pt1, pt2, color, thickness)
# pt1: начальная точка
# pt2: конечная точка
# color: цвет линии
# thickness: толщина линии

Пример:


# Загружаем изображение
image = cv2.imread("example8.jpg")

if image is None:
    print("Ошибка: изображение не загружено.")
else:
    color = (255, 0, 0)  # Синий цвет
    thickness = 2
    pt1 = (100, 100)
    pt2 = (500, 500)
    cv2.line(image, pt1, pt2, color, thickness)
    # pt1: координаты верхнего левого угла
    # pt2: координаты нижнего правого угла
    # color: цвет (например, (0, 255, 0) для зеленого)
    # thickness: толщина линии (-1 для заливки)

    # Сохраняем изображение
    cv2.imwrite("gygy.jpg", image)

    # Отображаем результат с использованием cv2_imshow
