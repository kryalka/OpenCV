Основные компоненты OpenCV

Библиотека OpenCV предоставляет широкий спектр функций для работы с изображениями и видео. Ниже рассмотрим основные возможности: загрузка, отображение и сохранение изображений.

Загрузка изображений

Для загрузки изображений используется функция cv2.imread(), которая преобразует изображение в массив NumPy.

Синтаксис cv2.imread()

cv2.imread(filename, flags)
  • cv2.IMREAD_COLOR — загружает цветное изображение (по умолчанию).
  • cv2.IMREAD_GRAYSCALE — загружает изображение в оттенках серого.
  • cv2.IMREAD_UNCHANGED — загружает изображение с альфа-каналом.

Примеры загрузки

Цветное изображение

import cv2
image = cv2.imread('example1.jpg', cv2.IMREAD_COLOR)
cv2.imshow('Цветное изображение', image)
cv2.waitKey(0)
cv2.destroyAllWindows()

Изображение в оттенках серого

gray_image = cv2.imread('example1.jpg', cv2.IMREAD_GRAYSCALE)
cv2.imshow('Оттенки серого', gray_image)
cv2.waitKey(0)
cv2.destroyAllWindows()

Отображение изображений

Для отображения изображений используется cv2.imshow(). В случае работы с Google Colab или Jupyter Notebook рекомендуется использовать Matplotlib.

import cv2
import matplotlib.pyplot as plt

image = cv2.imread('example1.jpg')
image_rgb = cv2.cvtColor(image, cv2.COLOR_BGR2RGB)
plt.imshow(image_rgb)
plt.axis('off')
plt.show()

Сохранение изображений

OpenCV позволяет сохранять обработанные изображения с помощью функции cv2.imwrite(). Например:

cv2.imwrite('output.jpg', image)

Для JPEG можно задать качество:

cv2.imwrite('output_90.jpg', image, [cv2.IMWRITE_JPEG_QUALITY, 90])

Сохранение с альфа-каналом:

cv2.imwrite('output_with_alpha.png', image_with_alpha)
